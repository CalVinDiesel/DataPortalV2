Manage Showcase

Manage which locations appear in the landing page showcase (tiles section). Use Sync from locations.json to add all locations from data/locations.json (same source as the map pins). The showcase is independent of the map: if you delete a pin from the overview map, it stays in the showcase until you remove it here.
