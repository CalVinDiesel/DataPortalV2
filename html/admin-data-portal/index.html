Quick info

This admin portal is linked to the database (PostgreSQL when configured). Use Add 3D Model to register new tileset URLs so they appear on the public overview map. Use Client Uploads to handle client requests for custom image-to-3D processing: clients submit their own drone images via the upload page; you process them and deliver the resulting 3D model back to the client (paid service; not added to the map).
